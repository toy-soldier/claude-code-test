# Claude Code Test

A collection of browser-based games built with plain HTML, CSS, and JavaScript. No build tools, no frameworks, no server required — just open a file in your browser and play.

## Games

### Tic-Tac-Toe (`tictactoe.html`)
Classic Tic-Tac-Toe with an unbeatable AI opponent.
- Play vs Computer or vs a friend
- Computer uses the **minimax algorithm** — it never loses
- X always goes first; computer plays O
- Score tracking per session

### Snake (`snake.html`)
Nokia-style Snake game with a retro Game Boy aesthetic.
- Classic snake gameplay on a wide board (36×20)
- **High score persists** across sessions (saved in localStorage)
- Score display shows current score and all-time best
- Control with arrow keys or on-screen buttons
- Press **OK** (or Enter) to start/restart

## How to Run

Just open the HTML file directly in any modern browser:

```
tictactoe.html
snake.html
```

No installation, no dependencies, no server needed.
